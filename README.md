Модуль **Flute CMS** для подключения [Plausible Analytics](https://plausible.io/) — веб-аналитики: просмотры, источники трафика, входные страницы и остальное, что даёт дашборд Plausible.

**Требования:** Flute `>= 1.0.3`

## Первичная настройка

1. Установите модуль во Flute
2. В админке откройте **Plausible Analytics** (`/admin/plausible-analytics`).
3. В Plausible в настройках сайта скопируйте **готовый сниппет** (блок с `<script …>`) и **вставьте его целиком** в поле сниппета в модуле.
4. Включите трекинг переключателем и сохраните.
